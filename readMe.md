#### node koa

kos 中间件 类似洋葱原理，一层层执行， 从外到里，然后在从里到外，注意每个中间件都要 next，没有 next 的话，后续中间件不会执行

#### MVC

- controller 控制业务逻辑
- service 控制数据交互
- middleware 引入常用中间件 或者 公共方法
